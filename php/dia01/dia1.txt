Cliente y Servidor

	- En Internet es una red 
	- La comunicación es de un cliente
	  a un servidor
	- Un cliente hace peticiones al servidor
	- El servidor atiende las peticiones    
	  de los todos los clientes.
	- El protocolo es el lenguaje de comunicación
	  entre el cliente y el servidor

	Páginas web:
	- Protocolos: HTTP 80, HTTPS 443
	- Cliente: Chrome, Firefox, Edge, Opera, Safari
	- Se envía: HTML, CSS, 
	- Servidor: Apache, EnginX, IIS
	
	FTP:
	- Protocolos: FTP 21, SFTP 22
	- Cliente: FileZilla Client
	- Servidor: FileZilla Server
